# AI Email Reply Chrome Extension

## Overview
AI-powered system that generates email replies using Gemini AI API.

## Features
- Automated email reply generation
- Context-aware responses
- Chrome Extension integration with Gmail
- Dynamic DOM handling using MutationObserver

## Tech Stack
- Backend: Spring Boot
- Frontend: JavaScript (Chrome Extension)
- API: Gemini AI

## Architecture
Chrome Extension → Spring Boot → Gemini AI → Response

## How to Run
1. Run Spring Boot backend
2. Load Chrome Extension
3. Use inside Gmail
